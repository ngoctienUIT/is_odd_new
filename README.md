# is_odd_new

Minimalist and elegant API that returns `true` if the given number or string is odd.

## Usage

```dart
import 'package:is_odd_new/is_odd_new.dart';

//int
isOdd(0);
//=> false
isOdd(1);
//=> true
isOdd(2);
//=> false
isOdd(3);
//=> true

//double
isOdd(0.0);
//=> false
isOdd(1.0);
//=> true
isOdd(2.0);
//=> false
isOdd(3.0);
//=> true

//String
isOdd("0");
//=> false
isOdd("1");
//=> true
isOdd("2");
//=> false
isOdd("3");
//=> true
```
